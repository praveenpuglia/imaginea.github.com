# Imaginea GitHub

This page lists some of our github projects and gives bird's eyeview of the technology and functionality that they cater to.

Usage:
=====

1) Clone or fork this repository, 
    Make sure you have node.js installed version 5+
    Make sure you have NPM installed version 3+
    
2) Run npm install to install dependencies

3) Run npm start to fire up lite dev server, which will launch the application in http://localhost:8000/

    if you want to use other port, open ls-config.js file, then change port.
    
4) Use npm run watch for live reloading

4) Finally for production builds, run 'npm run build'.
